# Primordial Ooze

A single-page PWA for the Primordial Ooze home group in Roseville, CA. It turns a public Google Sheet phone list into a milestone-first rolodex with call, text, contact export, sobriety odometers, and tier-based metal accents.

## Files

- `index.html` - app UI and Google Sheets loader
- `manifest.webmanifest` - PWA install metadata
- `service-worker.js` - app shell cache for install/offline behavior
- `ooze-header.png` - header artwork and app icon source
- `icon-192.svg`, `icon-512.svg` - generated fallback PWA icons

## Hosting

This app is static and can be hosted on GitHub Pages from the repository root.
